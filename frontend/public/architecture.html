Network

UniFi UDM
Edge Gateway and Switch Fabric
Ubiquiti Dream Machine handling the WAN uplink and inside-LAN switching for the matrix VLAN. Forwards traffic to PA-220 for inspection, then back out via untrust uplink.
  • WAN ingress and DHCP
  • Matrix VLAN switching
  • SNMP polled by Prometheus
  • Direct K3s pod egress via Pi2 eth2
UDM SNMP VLAN
PA-220 Firewall
reveal-fw, PAN-OS 10.2.10-h7
Palo Alto next-gen firewall enforcing four security zones with App-ID, threat prevention, and traffic logging.
Zone Interface Subnet
untrust eth1/7 UDM uplink
dmz-mgmt eth1/1 192.168.21.0/24
dmz-services eth1/6 192.168.20.0/24
dmz-security eth1/2, eth1/5 192.168.22.0/24
App-ID Threat Prevention Syslog to Vector
Tailscale Mesh
Zero-Trust Overlay
WireGuard-based mesh VPN connecting all hosts across the four security zones. Used for cross-zone traffic that PA-220 blocks by default.
  • Pi0, Pi1, Pi2, RV2, ThinkStation
  • MagicDNS for hostname routing
  • ACL-controlled access
  • Bypasses cross-zone PA-220 rules
WireGuard MagicDNS ACLs

Security Infrastructure

🛡️
Suricata IDS
Orange Pi RV2, RISC-V Edge Sensor
Network intrusion detection on RISC-V hardware with LLM-powered alert triage for intelligent filtering.
  • 74,096 rules (ET Open + 6 sources, daily auto-update)
  • Monitors dmz-security traffic (end0)
  • EVE JSON → rsyslog → Vector → Loki
  • Qwen2 1.5B LLM filters severity 3 alerts
  • Auto-escalate severity 1-2 to Telegram
Suricata 7.0.3 Qwen2 INT4 RISC-V Ky X1 8GB RAM

SentinelNet
K3s Pod, Exfiltration Detection
ML-based data exfiltration detection running on the K3s cluster alongside GuardQuote.

  • Integrates with Wazuh SIEM
  • Real-time alerting
